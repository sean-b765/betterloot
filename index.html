Rust is a survival game created by Facepunch. This tool helps server owners who have the Oxide plugin BetterLoot installed. You can edit the loot tables for each container easily and search for items here, without having to go through 4000 lines of text!
The containers without mouse-over image previews are not in Rust as of May 2021 (as they have no in-game prefabs)

The custom loot table used by BetterLoot is located at: /oxide/data/BetterLoot/LootTable.json
